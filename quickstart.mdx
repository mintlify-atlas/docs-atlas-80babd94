Get up and running with JS Beautify quickly. Choose your preferred runtime below.

<Tabs>
  <Tab title="Node.js Library">
    <Steps>
      <Step title="Install">
        ```bash
        npm install js-beautify
        ```
      </Step>
      <Step title="Beautify JavaScript">
        <CodeGroup>
          ```js ESM
          import beautify from 'js-beautify';

          const ugly = 'function sayHello(name){if(name){console.log("Hello, "+name);}else{console.log("Hello, World!");}}';
          const result = beautify.js(ugly, { indent_size: 2 });
          console.log(result);
          ```

          ```js CommonJS
          const beautify = require('js-beautify');

          const ugly = 'function sayHello(name){if(name){console.log("Hello, "+name);}else{console.log("Hello, World!");}}';
          const result = beautify.js(ugly, { indent_size: 2 });
          console.log(result);
          ```
        </CodeGroup>

        Output:
        ```js
        function sayHello(name) {
          if (name) {
            console.log("Hello, " + name);
          } else {
            console.log("Hello, World!");
          }
        }
        ```
      </Step>
      <Step title="Beautify CSS">
        ```js
        import beautify from 'js-beautify';

        const ugly = '.container{display:flex;flex-direction:column;}.header{background:#fbc826;color:#2a2a2a;}';
        const result = beautify.css(ugly, { indent_size: 2 });
        console.log(result);
        ```

        Output:
        ```css
        .container {
          display: flex;
          flex-direction: column;
        }

        .header {
          background: #fbc826;
          color: #2a2a2a;
        }
        ```
      </Step>
      <Step title="Beautify HTML">
        ```js
        import beautify from 'js-beautify';

        const ugly = '<html><head><title>Hello</title></head><body><h1>Hello World</h1><p>Welcome.</p></body></html>';
        const result = beautify.html(ugly, { indent_size: 2 });
        console.log(result);
        ```

        Output:
        ```html
        <html>
          <head>
            <title>Hello</title>
          </head>

          <body>
            <h1>Hello World</h1>
            <p>Welcome.</p>
          </body>
        </html>
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="CLI">
    <Steps>
      <Step title="Install globally">
        ```bash
        npm install -g js-beautify
        ```
      </Step>
      <Step title="Beautify a file">
        ```bash
        # Print beautified output to stdout
        js-beautify foo.js

        # Replace file in-place
        js-beautify -r foo.js

        # Write to a different file
        js-beautify foo.js -o foo-beautified.js
        ```
      </Step>
      <Step title="Beautify CSS and HTML">
        ```bash
        # Using dedicated commands
        css-beautify styles.css -r
        html-beautify index.html -r

        # Using the --type flag
        js-beautify --type css styles.css
        js-beautify --type html index.html
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Python">
    <Steps>
      <Step title="Install">
        ```bash
        # For JavaScript
        pip install jsbeautifier

        # For CSS
        pip install cssbeautifier
        ```
      </Step>
      <Step title="Beautify JavaScript">
        ```python
        import jsbeautifier

        # Beautify a string
        res = jsbeautifier.beautify('function foo(){return 42;}')
        print(res)

        # Beautify with options
        opts = jsbeautifier.default_options()
        opts.indent_size = 2
        res = jsbeautifier.beautify('function foo(){return 42;}', opts)
        print(res)
        ```
      </Step>
      <Step title="Beautify a file">
        ```python
        import jsbeautifier

        res = jsbeautifier.beautify_file('foo.js')
        print(res)
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Tip>
  CLI option names use dashes (e.g. `--indent-size`) while library option names use underscores (e.g. `indent_size`). They refer to the same options.
</Tip>
