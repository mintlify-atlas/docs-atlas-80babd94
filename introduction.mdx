JS Beautify is an open-source code formatter that handles JavaScript, CSS, and HTML. It re-indents minified or uglified code into a readable format, unpacks scripts packed by Dean Edward's popular packer, and can partially deobfuscate scripts processed by [javascript-obfuscator](https://github.com/javascript-obfuscator/javascript-obfuscator).

Try it in your browser at [beautifier.io](https://beautifier.io/).

## Key capabilities

**Script unpacking** — Detects and unpacks scripts packed by Dean Edward's popular packer, and partially deobfuscates output from javascript-obfuscator.

**Beautify directives** — Control the beautifier from inside your source files using inline comments. Use `/* beautify ignore:start */` to skip sections the beautifier cannot parse, or `/* beautify preserve:start */` to retain existing formatting for valid code sections.

**EditorConfig support** — Pass `--editorconfig` (or set `editorconfig: true` in options) and JS Beautify reads your `.editorconfig` file to align formatting rules with your editor settings.

**Language-specific overrides** — Apply different indentation and formatting rules per language in a single `.jsbeautifyrc` config file. Embedded `<script>` and `<style>` blocks inside HTML can inherit their own rules.

**Multiple runtimes** — Runs in Node.js (>=14), Python, and directly in the browser via CDN script tags.
