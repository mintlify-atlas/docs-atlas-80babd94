## What is JS Beautify?

JS Beautify reformats and re-indents JavaScript, CSS, and HTML code. It handles minified or uglified code, unpacks scripts packed by Dean Edward's packer, and can partially deobfuscate scripts processed by javascript-obfuscator.

## Multiple ways to use it

<Tabs>
  <Tab title="Node.js Library">
    ```bash
    npm install js-beautify
    ```

    ```js
    import beautify from 'js-beautify';

    const result = beautify.js('function foo(){return 42;}', { indent_size: 2 });
    console.log(result);
    // function foo() {
    //   return 42;
    // }
    ```
  </Tab>
  <Tab title="CLI">
    ```bash
    npm install -g js-beautify
    js-beautify foo.js
    ```
  </Tab>
  <Tab title="Python">
    ```bash
    pip install jsbeautifier
    ```

    ```python
    import jsbeautifier
    res = jsbeautifier.beautify('function foo(){return 42;}')
    ```
  </Tab>
  <Tab title="Browser">
    ```html
    <script src="https://cdnjs.cloudflare.com/ajax/libs/js-beautify/1.15.4/beautify.min.js"></script>
    <script>
      var result = js_beautify('function foo(){return 42;}', { indent_size: 2 });
    </script>
    ```
  </Tab>
</Tabs>

## Key features

<AccordionGroup>
  <Accordion title="Highly configurable">
    Control indentation size and character, brace style, line wrapping, operator position, and dozens of other formatting preferences through CLI flags, a `.jsbeautifyrc` file, or programmatic options.
  </Accordion>
  <Accordion title="EditorConfig integration">
    JS Beautify automatically reads your `.editorconfig` file to align formatting rules with your editor settings — no duplicate configuration needed.
  </Accordion>
  <Accordion title="Language-specific overrides">
    Apply different formatting rules for JavaScript, CSS, and HTML within a single `.jsbeautifyrc` config. Embedded scripts and styles inside HTML inherit their own rules.
  </Accordion>
  <Accordion title="Beautify directives">
    Use `/* beautify ignore:start */` and `/* beautify preserve:start */` directives in your source files to protect sections from being reformatted.
  </Accordion>
  <Accordion title="Templating language support">
    Handles templating languages including Django, ERB, Handlebars, PHP, Smarty, and Angular inside HTML files without corrupting template syntax.
  </Accordion>
  <Accordion title="Unpacker support">
    Detects and unpacks scripts packed by Dean Edward's popular packer, and partially deobfuscates scripts processed by javascript-obfuscator.
  </Accordion>
</AccordionGroup>
